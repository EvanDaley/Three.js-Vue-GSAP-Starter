### Deployment
This uses the gh-pages npm package to manage deployment.

To deploy a new version:
```
# Update the build
npm run build

# Trigger a commit to the github pages branch
npm run deploy
```

For more information on deployment, see: https://www.npmjs.com/package/gh-pages
